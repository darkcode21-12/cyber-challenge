
# 🌐 OSI Model (Open Systems Interconnection)

The **OSI Model** is used for communication between different devices. It is a **7-layer conceptual framework** designed to:

* Standardize communication between systems
* Help understand how data flows across a network

---

## 📊 The 7 Layers (Top → Bottom)

```
7. Application  
6. Presentation  
5. Session  
4. Transport  
3. Network  
2. Data Link  
1. Physical  
```

---

## 🔝 7. Application Layer

* Interfaces directly with applications
* Provides network services to software

### 🔧 Examples:

* **SMTP** → Sending and receiving email
* **HTTP** → Data communication on the World Wide Web

---

## 🎨 6. Presentation Layer

* Translates data between formats
* Handles:

  * Encryption
  * Decryption
  * Data formatting

📌 Ensures data is readable on both ends

---

## 🔗 5. Session Layer

* Manages connections between devices
* Controls communication:

  * Who sends data
  * When data is sent

### 🔧 Examples:

* API sessions
* Login sessions

---

## 🚚 4. Transport Layer

* Ensures reliable data transfer
* Handles:

  * Flow control
  * Error detection

### 🔧 Examples:

* **TCP (Transmission Control Protocol)**

  * Reliable delivery
  * Used in file transfer and web browsing

* **UDP (User Datagram Protocol)**

  * Fast delivery
  * “Best-effort” (speed over reliability)

---

## 🌍 3. Network Layer

* Determines the best path for data
* Uses **logical addressing**

### 📌 Logical Addressing:

A virtual identifier assigned by software (not hardware)

### 🔧 Example:

* **IP (Internet Protocol)**

---

## 🔌 2. Data Link Layer

* Handles communication within the same network
* Uses **physical addressing (MAC address)**

### 🔧 Example:

* Switches

---

## ⚡ 1. Physical Layer

* Transmits raw binary data (bits)
* Uses physical signals such as:

  * Electrical voltage
  * Light signals
  * Radio waves

---

## 🧠 Summary

The OSI Model helps break down complex networking into manageable layers, making it easier to:

* Understand data communication
* Troubleshoot network issues
* Design secure systems
