# 🌐 30 Common Ports Used in Daily Life



## 🔑 Core Web & Internet

| Port | Protocol | Description |
|------|--------|-------------|
| 80   | HTTP   | Web traffic (non-secure websites) |
| 443  | HTTPS  | Secure web traffic (SSL/TLS) |
| 8080 | HTTP Alt | Alternative web server port |

---

## 🔐 Remote Access

| Port | Protocol | Description |
|------|--------|-------------|
| 22   | SSH    | Secure remote login (Linux servers) |
| 23   | Telnet | Insecure remote login |
| 3389 | RDP    | Remote Desktop (Windows) |

---

## 📁 File Transfer

| Port | Protocol | Description |
|------|--------|-------------|
| 20   | FTP Data | File transfer data |
| 21   | FTP Control | File transfer control |
| 22   | SFTP   | Secure file transfer over SSH |

## ⚡ Quick Notes

- Ports = **process communication endpoints**
- Same port can use:
  - TCP (reliable)
  - UDP (fast)

### 🔥 Most Important Ports:
- 80 / 443 → Web
- 22 → SSH
- 53 → DNS
- 3389 → RDP
